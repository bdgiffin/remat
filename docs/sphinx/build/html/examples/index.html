ExamplesÔÉÅ

The following series of examples provide an interactive demonstration of the reversible physics capabilities currently implemented in REMAT. All examples make use of the WebAssembly build of REMAT in combination with pygbag to enable native execution and real-time animation in the web browser. The resulting graphical demos were partially inspired by the interactive Exact Bitwise Reversible Integrator created by Jos Stam.[1]

For all examples:

  • → (right arrow key): Advance the interactive simulation forward in time.

  • ← (left arrow key): Advance the interactive simulation backward in time.

Examples:

  • Bouncing Ball
  • Contacting Blocks
  • Potential Well
  • Seismic Wave
  • Triple Pendulum
  • Viscoplastic Truss
[1]

Jos Stam. An exact bitwise reversible integrator. 2022. URL: https://arxiv.org/abs/2207.07695, arXiv:2207.07695.
